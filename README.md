# WhatsApp RAG Assistant (SiteWA Bot)

A SaaS platform enabling businesses to deploy AI-powered WhatsApp chatbots with Retrieval-Augmented Generation (RAG) capabilities. The bot answers queries accurately based on website content and policies, reducing support tickets.

## Features

- **WhatsApp Business Cloud API Integration**: Full compliance with Meta's policies
- **Advanced RAG Engine**: Powered by LlamaIndex for accurate knowledge retrieval
- **Multilingual Support**: Handles 50+ languages with auto-detection
- **Voice Processing**: Speech-to-text and optional text-to-speech capabilities
- **Multi-Tenant Architecture**: Complete data isolation between businesses
- **Easy Website Integration**: One-click embeddable chat widget

## Architecture

- **Backend**: Python with FastAPI
- **RAG Framework**: LlamaIndex
- **Vector Database**: Pinecone or Qdrant
- **Frontend**: Next.js dashboard
- **Voice Processing**: OpenAI Whisper for STT
